# binm.atClient

A framework to communicate with https://binm.at via TypeScript

Binmat Rules: https://github.com/DrizzlyBear/binmat_rules

# Requirements

* npm
* NodeJS
* TypeScript
* ts-node

Make sure to have a credentials.json in the top folder like this:
[
    {
        "user": "myFirstUser",
        "pass": "Pass1"
    },

    {
        "user": "mySecondUser",
        "pass": "Pass2"
    }
]

# How to start

ts-node main
